# software implementation
Use 6 softwares to check posit implemenation and accuracy.  
This check allows us to algorithmically validate posit.

These implementations are :
+ Software Emulation :
  - Ceralaine
  - Fastsigmoids
  - Posit-GCC
  - Softposit
+ RacEr
+ CUDA
+ Flopoco (FPGA implementation of Posit)
